## 茉莉QQ机器人（简称MQR）

### 介绍
采用mirai的Android协议实现的QQ机器人服务，可以通过[web页面](https://github.com/xggz/mqr-web)控制机器人的运行。

### 软件架构
`MQR` 使用Java语言开发（jdk 8），技术栈：`Spring Boot` + `JWT` + `Mybatis Plus` + `Sqlite`，查看和运行源码的 IDE 需安装 lombok 插件。
